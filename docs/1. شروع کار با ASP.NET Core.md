
در این بخش با نحوه ایجاد، اجرا و ویرایش یک برنامه **Blazor Web App** در **ASP.NET Core** آشنا می‌شویم. برای انجام این کار از **.NET CLI** (رابط خط فرمان دات‌نت) استفاده خواهیم کرد.

**Blazor** یکی از فریم‌ورک‌های رابط کاربری (Front-End) در اکوسیستم .NET است که امکان ساخت برنامه‌های وب تعاملی را با استفاده از زبان **C#** فراهم می‌کند. این فریم‌ورک از **رندر سمت سرور (Server-Side Rendering)** و همچنین **تعامل سمت کاربر (Client Interactivity)** در قالب یک مدل برنامه‌نویسی واحد پشتیبانی می‌کند.

در پایان این بخش خواهید توانست:

- یک پروژه Blazor ایجاد کنید.
    
- پروژه را اجرا کنید.
    
- تغییرات موردنظر را روی پروژه اعمال کنید.
    
- اجرای برنامه را متوقف کنید.
    

---

## پیش‌نیازها

قبل از شروع، باید آخرین نسخه **.NET SDK** را روی سیستم خود نصب کنید.

.NET SDK شامل ابزارهای لازم برای ایجاد، اجرا، کامپایل و مدیریت پروژه‌های دات‌نت است و همچنین ابزار **.NET CLI** را در اختیار شما قرار می‌دهد.

---

## ایجاد پروژه Blazor

ابتدا **Command Prompt**، **PowerShell** یا **Terminal** را باز کنید و وارد پوشه‌ای شوید که قصد دارید پروژه در آن ایجاد شود.

سپس دستور زیر را اجرا کنید:

```bash
dotnet new blazor -o BlazorSample
```

### بررسی دستور

- **dotnet new** : ایجاد یک پروژه جدید
    
- **blazor** : انتخاب قالب پروژه Blazor
    
- **-o BlazorSample** : ایجاد پروژه داخل پوشه‌ای با نام **BlazorSample**
    

پس از اجرای این دستور، ساختار اولیه پروژه به‌صورت خودکار ایجاد خواهد شد.

---

## اجرای پروژه

ابتدا وارد پوشه پروژه شوید:

```bash
cd BlazorSample
```

سپس برنامه را اجرا کنید:

```bash
dotnet watch
```

دستور **dotnet watch** علاوه بر اجرای برنامه، تغییرات فایل‌های پروژه را نیز زیر نظر می‌گیرد. هر زمان که فایل‌ها را ذخیره کنید، برنامه بدون نیاز به اجرای مجدد، به‌صورت خودکار به‌روزرسانی می‌شود.

پس از اجرای دستور، مرورگر به‌صورت خودکار باز شده و صفحه اصلی برنامه نمایش داده می‌شود.

---

## آشنایی با برنامه

در منوی کناری برنامه گزینه‌ای با عنوان **Counter** وجود دارد.

با ورود به این صفحه، دکمه‌ای با عنوان **Click me** مشاهده می‌کنید.

هر بار که روی این دکمه کلیک کنید، مقدار شمارنده یک واحد افزایش پیدا می‌کند.

برای مثال:

قبل از کلیک:

```text
Current count: 0
```

بعد از اولین کلیک:

```text
Current count: 1
```

---

# بررسی فایل Counter.razor

صفحه Counter توسط فایل زیر ایجاد می‌شود:

```text
Components/Pages/Counter.razor
```

فایل‌های با پسوند **.razor** مخصوص Blazor هستند و می‌توان در آن‌ها کدهای **HTML** و **C#** را در کنار یکدیگر نوشت.

به این فایل‌ها **Razor Component** گفته می‌شود.

نمونه کد این فایل به شکل زیر است:

```razor
@page "/counter"

<PageTitle>Counter</PageTitle>

<h1>Counter</h1>

<p role="status">Current count: @currentCount</p>

<button class="btn btn-primary" @onclick="IncrementCount">
    Click me
</button>

@code {
    private int currentCount = 0;

    private void IncrementCount()
    {
        currentCount++;
    }
}
```

اکنون بخش‌های مختلف این کد را بررسی می‌کنیم.

---

## تعیین مسیر صفحه

```razor
@page "/counter"
```

دستور **@page** مشخص می‌کند که این کامپوننت با چه آدرسی در مرورگر در دسترس باشد.

در این مثال، با مراجعه به مسیر زیر، صفحه Counter نمایش داده می‌شود:

```text
/counter
```

---

## تعیین عنوان صفحه

```razor
<PageTitle>Counter</PageTitle>
```

تگ **PageTitle** عنوان صفحه را مشخص می‌کند.

متن داخل این تگ در زبانه (Tab) مرورگر نمایش داده می‌شود.

---

## نمایش عنوان اصلی

```html
<h1>Counter</h1>
```

این تگ یک عنوان سطح اول (Heading 1) در صفحه ایجاد می‌کند.

---

## نمایش مقدار شمارنده

```razor
<p role="status">
    Current count: @currentCount
</p>
```

در این قسمت مقدار متغیر **currentCount** داخل صفحه نمایش داده می‌شود.

علامت **@** به Razor اعلام می‌کند که مقدار یک متغیر C# را در خروجی HTML قرار دهد.

اگر مقدار متغیر برابر **5** باشد، خروجی به شکل زیر خواهد بود:

```html
<p>Current count: 5</p>
```

---

## دکمه افزایش شمارنده

```razor
<button class="btn btn-primary"
        @onclick="IncrementCount">
    Click me
</button>
```

ویژگی **@onclick** یک رویداد (Event) را تعریف می‌کند.

هر بار که کاربر روی دکمه کلیک کند، متد **IncrementCount()** اجرا خواهد شد.

---

## بلوک @code

تمام منطق برنامه داخل بلوک **@code** قرار می‌گیرد.

```csharp
@code
{
    private int currentCount = 0;

    private void IncrementCount()
    {
        currentCount++;
    }
}
```

این بلوک شامل کدهای C# مربوط به کامپوننت است.

---

### تعریف متغیر

```csharp
private int currentCount = 0;
```

در این خط یک متغیر از نوع **int** تعریف شده است.

مقدار اولیه این متغیر **0** است.

---

### متد IncrementCount

```csharp
private void IncrementCount()
{
    currentCount++;
}
```

این متد هر بار که اجرا شود، مقدار متغیر **currentCount** را یک واحد افزایش می‌دهد.

دستور

```csharp
currentCount++;
```

معادل عبارت زیر است:

```csharp
currentCount = currentCount + 1;
```

---

# اعمال تغییر در برنامه

فرض کنید می‌خواهید با هر بار کلیک روی دکمه، شمارنده به جای یک واحد، **ده واحد** افزایش پیدا کند.

کافی است کد زیر را تغییر دهید.

قبل از تغییر:

```csharp
currentCount++;
```

بعد از تغییر:

```csharp
currentCount += 10;
```

عبارت بالا معادل کد زیر است:

```csharp
currentCount = currentCount + 10;
```

پس از ذخیره فایل، نیازی به اجرای مجدد برنامه نیست. چون پروژه با دستور **dotnet watch** اجرا شده است، تغییرات به‌صورت خودکار در مرورگر اعمال می‌شوند.

اگر مقدار شمارنده قبل از تغییر برابر **1** باشد، پس از یک بار کلیک روی دکمه مقدار آن به **11** خواهد رسید و در هر کلیک بعدی نیز 10 واحد به آن اضافه می‌شود.

## جمع‌بندی

در این بخش با مراحل ایجاد و اجرای اولین برنامه **Blazor** آشنا شدیم و یاد گرفتیم چگونه:

- یک پروژه جدید با استفاده از **.NET CLI** ایجاد کنیم.
    
- پروژه را با دستور **dotnet watch** اجرا کنیم.
    
- ساختار فایل **Counter.razor** را بررسی کنیم.
    
- از دستور **@page** برای تعیین مسیر صفحه استفاده کنیم.

این مثال یکی از ساده‌ترین نمونه‌های Blazor است، اما مفاهیم مهمی مانند **کامپوننت‌ها (Components)**، **رویدادها (Events)**، **ترکیب HTML و C# با Razor** و **به‌روزرسانی خودکار رابط کاربری** را معرفی می‌کند؛ مفاهیمی که پایه توسعه برنامه‌های حرفه‌ای با ASP.NET Core هستند.
