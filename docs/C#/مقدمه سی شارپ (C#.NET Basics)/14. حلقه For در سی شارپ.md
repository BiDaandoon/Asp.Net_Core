## **حلقه For در سی شارپ به همراه مثال**

در این مقاله، قصد دارم **C#** حلقه For را در زبان به همراه مثال بررسی کنم. در پایان این مقاله، شما خواهید فهمید که حلقه For چیست و چه زمانی و چگونه باید از حلقه For در زبان C# به همراه مثال استفاده کرد.

##### **حلقه for در سی شارپ:**

حلقه for یکی از رایج‌ترین حلقه‌های مورد استفاده در زبان سی‌شارپ است. اگر تعداد دفعات را بدانیم، می‌خواهیم مجموعه‌ای از دستورات یا گزاره‌ها را اجرا کنیم، باید از حلقه for استفاده کنیم. حلقه for به عنوان یک حلقه شمارنده شناخته می‌شود. هر زمان که شمارش برای تکرار مطرح باشد، باید از حلقه for استفاده کنیم.

بیایید یک مثال بزنیم و سعی کنیم با شمارش بفهمیم که منظور از آن چیست. فرض کنید در حال تهیه قهوه هستید. نمی‌دانید چگونه قهوه درست کنید؛ کسی به شما دستورالعمل داده است و شما از آنها پیروی می‌کنید. شما برای ۴ نفر قهوه درست می‌کنید.

حالا سوال این است که چقدر شکر باید اضافه کنید؟ شما دقیقاً نمی‌دانید چقدر شکر باید اضافه کنید. بنابراین، کاری که انجام خواهید داد این است که مقداری شکر اضافه می‌کنید و آن را مخلوط می‌کنید و بررسی می‌کنید که آیا قهوه شیرین است یا خیر، اگر به اندازه کافی شیرین نیست، دوباره مقداری شکر اضافه می‌کنید و مخلوط می‌کنید و به همین ترتیب تا زمانی که قهوه به اندازه کافی شیرین شود. بنابراین، چند بار باید شکر به قهوه اضافه کنید؟ این سوال جواب ثابتی ندارد زیرا شما شکر را به قهوه اضافه می‌کنید تا زمانی که به اندازه کافی شیرین شود. بنابراین، این بر اساس شمارش نیست. این بر اساس شرایط است. شرایط تا زمانی است که و مگر اینکه قهوه به اندازه کافی شیرین نباشد، شما به اضافه کردن شکر ادامه می‌دهید و آن را مخلوط می‌کنید.

حالا کسی به شما دستورالعملی می‌دهد که برای درست کردن قهوه برای چهار نفر، ۴ قاشق غذاخوری شکر اضافه می‌کند. خب، این بر اساس شمارش است. مثال‌های زیادی در زندگی روزمره وجود دارد. ما کارها را به تعداد دفعات ثابتی انجام می‌دهیم. بنابراین، وقتی مجبورید مراحل را بر اساس شمارش تکرار کنید، باید از حلقه for استفاده کنید.

##### **فلوچارت حلقه For**

نمودار زیر، فلوچارت حلقه for را نشان می‌دهد.

![فلوچارت حلقه for در سی شارپ](https://dotnettutorials.net/wp-content/uploads/2022/04/For-Loop-Flowchart-in-C.png "فلوچارت حلقه for در سی شارپ")

نمودار جریان شروع می‌شود. شروع با نماد بیضی نشان داده می‌شود. سپس شرط بررسی می‌شود. همانطور که قبلاً بحث شد، هر شرط دو خروجی دارد، یعنی درست و نادرست. اگر درست باشد چه اتفاقی خواهد افتاد و اگر نادرست باشد چه اتفاقی خواهد افتاد، باید بررسی کنیم.

فرض کنید شرط درست باشد، آنگاه تمام دستورات تعریف شده درون بلوک (درون بلوک حلقه for) اجرا خواهند شد. آیا پس از اجرای دستورات، حلقه پایان می‌یابد؟ خیر، پایان نمی‌یابد. پس از اجرای دستورات، یک بار دیگر شرط حلقه for بررسی می‌شود. تا زمانی که شرط حلقه درست باشد، همین فرآیند تکرار می‌شود. و هنگامی که شرط نادرست شود، حلقه پایان می‌یابد. این جریان اجرای حلقه for در C# است.

##### **سینتکس استفاده از حلقه For در زبان سی شارپ:**

حلقه **for** امکان اجرای دستورالعمل‌ها را برای مدت زمان مشخصی فراهم می‌کند و چهار مرحله دارد.

1. **مقداردهی اولیه حلقه**
2. **ارزیابی وضعیت**
3. **اجرای دستورالعمل**
4. **افزایش/کاهش**

###### **توضیح سینتکس حلقه for:**

1. **مقداردهی اولیه حلقه:** مقداردهی اولیه حلقه فقط یک بار در حین اجرای حلقه for اتفاق می‌افتد، به این معنی که بخش مقداردهی اولیه حلقه for فقط یک بار اجرا می‌شود. در اینجا، مقداردهی اولیه به این معنی است که باید متغیر شمارنده را مقداردهی اولیه کنیم.
2. **ارزیابی شرط:** شرط‌های حلقه for برای هر تکرار اجرا می‌شوند و اگر شرط درست باشد، دستورالعمل C# اجرا می‌شود و اگر شرط نادرست باشد، از حلقه خارج می‌شود.
3. **اجرای دستورالعمل:** پس از ارزیابی شرط، و در صورت درست بودن شرط، کنترل به بدنه حلقه منتقل می‌شود، یعنی بدنه حلقه اجرا خواهد شد.
4. **افزایش/کاهش** : پس از اجرای بدنه حلقه، بخش افزایش/کاهش حلقه for اجرا خواهد شد و به محض اجرای بخش افزایش/کاهش، یعنی به محض افزایش و کاهش متغیر شمارنده، دوباره به مرحله ارزیابی شرط می‌رود.

##### **نکاتی که هنگام کار با حلقه for در سی شارپ باید به خاطر داشته باشید:**

1. وقتی با حلقه for کار می‌کنیم، فرآیند اجرا همیشه از بلوک مقداردهی اولیه شروع می‌شود. پس از بلوک مقداردهی اولیه، کنترل به بلوک شرط منتقل می‌شود. اگر شرط درست ارزیابی شود، کنترل به بلوک دستور منتقل می‌شود.
2. پس از اجرای بلوک دستور، کنترل به دستور افزایش/کاهش منتقل می‌شود، و از دستور افزایش/کاهش، به دستور شرط برمی‌گردد. تکرار همیشه اتفاق می‌افتد. **شرط اولیه** ، **بلوک دستور** و **دستور افزایش/کاهش.** بلوک مقداردهی اولیه فقط یک بار اجرا می‌شود، زمانی که برای اولین بار وارد حلقه می‌شویم.
3. وقتی با حلقه for کار می‌کنیم، همه چیز اختیاری است اما قرار دادن دو نقطه‌ویرگول (;;) اجباری است.
4. در حالی که ما با حلقه for کار می‌کنیم، اگر بخش شرط داده نشود، حلقه به دفعات نامحدود تکرار می‌شود زیرا بخش شرط آن را به عنوان غیر صفر جایگزین می‌کند. بنابراین همیشه درست است، مانند for(; 1; )
5. در حلقه for نیز فرآیند پیش‌بررسی رخ می‌دهد، یعنی قبل از اجرای بلوک دستور (بدنه حلقه for)، بخش شرط اجرا می‌شود.

##### **مثال چاپ اعداد از ۱ تا n با استفاده از حلقه For در سی شارپ:**

ابتدا، عدد ورودی را از کاربر می‌گیریم. این عددی است که از یک تا به بالا چاپ می‌شود. در اینجا، متغیر شمارنده را با عدد ۱ مقداردهی اولیه می‌کنیم زیرا می‌خواهیم عدد از ۱ چاپ شود. سپس باید بررسی کنیم که آیا متغیر شمارنده کمتر از مساوی عدد ورودی است که از کلمه کلیدی/کاربر گرفته‌ایم یا خیر. اگر شرط درست باشد، مقدار شمارنده را چاپ می‌کنیم و سپس متغیر شمارنده را ۱ واحد افزایش می‌دهیم. پس از به‌روزرسانی متغیر شمارنده، باید دوباره شرط را بررسی کنیم و اگر شرط درست باشد، همین فرآیند را تکرار می‌کنیم. به محض اینکه شرط نادرست شود، حلقه متوقف می‌شود و کنترل از حلقه خارج می‌شود.

```csharp
using System;

namespace ControlFlowDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.Write("Enter one Integer Number:");
            int number = Convert.ToInt32(Console.ReadLine());
            for (int counter = 1; counter <= number; counter++)
            {
                Console.WriteLine(counter);
            }
            Console.ReadKey();
        }
    }
}
```

```csharp
using System;

namespace ControlFlowDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.Write("Enter one Integer Number:");
            int number = Convert.ToInt32(Console.ReadLine());
            int counter = 1; //Initialization
            for (; counter <= number;)
            {
                Console.WriteLine(counter);
                counter++; //Updation
            }
            Console.ReadKey();
        }
    }
}
```

##### **حلقه بی‌نهایت در سی شارپ:**

در مثال قبلی، دیدیم که می‌توانیم عبارت افزایش/کاهش را در قسمت بدنه قرار دهیم. اما اگر هیچ عبارت افزایش/کاهشی در حلقه for یا در بدنه قرار ندهیم چه اتفاقی می‌افتد؟ این یک حلقه بی‌نهایت خواهد بود. حلقه بی‌نهایت، حلقه‌ای بی‌پایان است. برای درک بهتر، لطفاً به مثال زیر نگاهی بیندازید. در مثال زیر، ما عبارت افزایش/کاهش را نداریم، بنابراین حلقه for پایان نمی‌یابد و از این رو منجر به یک حلقه بی‌نهایت می‌شود.

```csharp
using System;

namespace ControlFlowDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            int number = 10;
            int i = 1;
            for (; i <= number; )
            {
                Console.WriteLine("Hello C#");
            }
            Console.ReadKey();
        }
    }
}
```

##### **آیا می‌توانیم حلقه for را بدون شرط در سی شارپ اجرا کنیم؟**

بله، می‌توانیم حلقه for را بدون شرط اجرا کنیم. و این یک حلقه بی‌نهایت خواهد بود. زیرا اگر هیچ شرط پایانی را در حلقه for ذکر نکنیم، حلقه for پایان نمی‌یابد. برای درک بهتر، لطفاً به مثال زیر نگاهی بیندازید.

```csharp
using System;

    {
        static void Main(string[] args)

                Console.WriteLine("Hello C#");
            }
        }
    }
}
```

```csharp
{
    class Program
    {
        static void Main(string[] args)

            {

                i++;
                if (i > 5)
                    break;
            }
            Console.ReadKey();
        }
    }
}
```

##### **حلقه for تو در تو در سی شارپ:**

وقتی یک حلقه for را درون بدنه حلقه for دیگری ایجاد می‌کنیم، گفته می‌شود که در زبان سی شارپ، حلقه for تودرتو ایجاد شده است. سینتکس استفاده از حلقه for تودرتو در زیر آمده است.

![حلقه for تو در تو در سی شارپ](https://dotnettutorials.net/wp-content/uploads/2022/04/nested-for-loop-in-c-language.png "حلقه for تو در تو در سی شارپ")

**نکته:** نکته‌ای که باید به خاطر داشته باشید این است که وقتی شرط حلقه داخلی ناموفق باشد، فقط حلقه داخلی را خاتمه می‌دهد. و وقتی شرط حلقه خارجی ناموفق باشد، حلقه خارجی را خاتمه می‌دهد.

##### **مثال برای درک حلقه‌های تو در تو در سی شارپ:**

در مثال زیر، ما یک حلقه for تو در تو ایجاد کرده‌ایم. حلقه for بیرونی ۵ بار اجرا می‌شود و برای هر تکرار حلقه for بیرونی، حلقه for داخلی ۱۰ بار اجرا می‌شود.

```csharp
using System;

namespace ControlFlowDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            for (int i = 1; i <= 5; i++) //Outer For Loop
            {
                Console.WriteLine($"Outer For Loop : {i}");
                for (int j = 1; j <= 10; j++) //Inner For Loop
                {
                    Console.Write($" {j}");
                }
                Console.WriteLine();
            }

            Console.ReadKey();
        }
    }
}
```

##### **برنامه‌ای برای وارد کردن یک عدد و بررسی اینکه آیا آن عدد کامل است یا خیر با استفاده از حلقه for در سی شارپ**

یک عدد کامل، عدد صحیح مثبتی است که با مجموع مقسوم‌علیه‌های مثبت خود، به جز خود عدد، برابر باشد. برای مثال، عدد ۶ مقسوم‌علیه‌های ۱، ۲ و ۳ دارد و ۱ + ۲ + ۳ = ۶، بنابراین ۶ یک عدد کامل است.

```csharp
using System;

namespace ControlFlowDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            int number, i, sum = 0;
            Console.Write("Enter a Number :");
            number = Convert.ToInt32(Console.ReadLine());

            for (i = 1; i <= number / 2; i++)
            {
                if (number % i == 0)
                    sum += i;
            }
            if (sum == number && number != 0)
                Console.WriteLine($"{number} is a Perfect Number");
            else
                Console.WriteLine($"{number} is not a Perfect Number");

            Console.ReadKey();
        }
    }
}
```

عدد آرمسترانگ عددی است که برابر با مجموع توان هر رقم در تعداد کل ارقام است. برای مثال، اعدادی مانند ۰، ۱، ۱۵۳، ۳۷۰، ۳۷۱، ۴۰۷، ۱۶۳۴، ۸۲۰۸، ۹۴۷۴ اعداد آرمسترانگ هستند. بیایید نگاهی به نمودار زیر بیندازیم که نحوه محاسبه عدد آرمسترانگ را نشان می‌دهد.

![برنامه‌ای برای بررسی اینکه آیا یک عدد، عدد آرمسترانگ است یا خیر، با استفاده از حلقه for به زبان سی شارپ](https://dotnettutorials.net/wp-content/uploads/2022/04/Program-to-check-whether-a-number-is-Armstrong-number-or-not-using-for-loop-C-Language.png "برنامه‌ای برای بررسی اینکه آیا یک عدد، عدد آرمسترانگ است یا خیر، با استفاده از حلقه for به زبان سی شارپ")

```csharp
using System;

namespace ControlFlowDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            int i = 0;
            int digitCount = 0;
            int[] digitArray = new int[10];
            double sum = 0;

            //Step1: Take the input
            Console.Write("Enter a Number : ");
            int number = int.Parse(Console.ReadLine());

            //Step3: Store the number in a temporary variable
            int temporaryNumber = number;

            //Step3: Find the total number of digits in number as well as
            //Store each digit in the digit array
            while (number > 0)
            {
                digitArray[i++] = number % 10;
                number = number / 10;
                digitCount++;
            }

            //Step4: Calculate the result
            for (i = 0; i < digitCount; i++)
            {
                sum += Math.Pow(digitArray[i], digitCount);
            }

            //Step5: Check whether it is prime number or not
            if (sum == temporaryNumber)
            {
                Console.WriteLine($"{temporaryNumber} is an Armstrong number");
            }
            else
            {
                Console.WriteLine($"{temporaryNumber} is not an Armstrong number");
            }

            Console.ReadLine();
        }
    }
}
```

##### **برنامه ای برای وارد کردن یک عدد و بررسی اینکه آیا عدد اول است یا خیر با استفاده از حلقه for در زبان سی شارپ**

عدد اول عددی است که باید بزرگتر از ۱ باشد و فقط بر ۱ و خودش تقسیم می‌شود. به عبارت دیگر، می‌توان گفت که اعداد اول نمی‌توانند بر اعداد دیگری غیر از خودشان و ۱ تقسیم شوند. برای مثال، ۲، ۳، ۵، ۷، ۱۱، ۱۳، ۱۷، ۱۹ و ۲۳ ... اعداد اول هستند.

```csharp
using System;

namespace ControlFlowDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            int number, i;
            Console.Write("Enter a Number :");
            number = Convert.ToInt32(Console.ReadLine());

            for (i = 2; i < number; i++)
            {
                if (number % i == 0)
                {
                    break;
                }
            }
            if (i == number && number >= 2)
            {
                Console.WriteLine($"{number} is a Prime Number");
            }
            else
            {
                Console.WriteLine($"{number} is not a Prime Number");
            }

            Console.ReadKey();
        }
    }
}
```

```csharp
            if (number >= 1)
            {
                Console.Write($"{number1} {number2}");
                temp = number1 + number2;

                for (; temp <= number;)
                {
                    Console.Write($" {temp}");
                    number1 = number2;
                    number2 = temp;
                    temp = number1 + number2;
                }
            }

                Console.WriteLine("please enter a number greater than zero");
```
