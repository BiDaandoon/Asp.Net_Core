## **دستور Continue در سی شارپ به همراه مثال**

در این مقاله، قصد دارم **دستور Continue را در** زبان سی شارپ با مثال بررسی کنم. در پایان این مقاله، شما خواهید فهمید که دستور Continue چیست و چه زمانی و چگونه می‌توان از دستور continue در زبان سی شارپ با مثال استفاده کرد.

##### **دستور Continue در زبان سی شارپ:**

در سی شارپ، continue یک کلمه کلیدی است. با استفاده از کلمه کلیدی continue، می‌توانیم از اجرای دستور در بدنه حلقه صرف نظر کنیم. مانند **دستور break** ، استفاده از دستور continue نیز اختیاری است، اما اگر می‌خواهید از آن استفاده کنید، می‌توانید آن را فقط در بدنه حلقه به کار ببرید.

اگر حداکثر تعداد تکرارها را بدانیم اما شرطی داشته باشیم و وقتی شرط برقرار باشد، باید از اجرای دستور در بدنه حلقه صرف نظر کنیم و اجرای حلقه را برای تکرار بعدی ادامه دهیم، و این در C# با استفاده از دستور continue در C# امکان‌پذیر است. دستور Continue در C# روشی مناسب برای شروع فوری تکرار بعدی حلقه‌های FOR، WHILE، Do While و for هر حلقه فراهم می‌کند.

دستور **BREAK** حلقه را خاتمه می‌دهد، در حالی که **دستور CONTINUE** فقط از تکرار فعلی حلقه صرف نظر می‌کند و به تکرار بعدی حلقه اجازه ادامه می‌دهد. دستور continue تقریباً همیشه با دستور if…else استفاده می‌شود.

دستور continue در سی شارپ بسیار شبیه به دستور break است، با این تفاوت که به جای خاتمه دادن به حلقه، از تکرار فعلی صرف نظر کرده و با تکرار بعدی ادامه می‌دهد. این بدان معناست که دستور continue از بقیه بدنه حلقه صرف نظر کرده و بلافاصله شرط حلقه را بررسی می‌کند. اگر شرط حلقه درست باقی بماند، اجرای حلقه ادامه می‌یابد.

**نحو: continue;**

##### **دستور Continue در زبان سی شارپ چگونه کار می‌کند؟**

حال، بیایید نحوه استفاده از دستور continue درون حلقه را درک کنیم و ببینیم که دستور continue دقیقاً در سی شارپ چگونه کار می‌کند. برای درک این موضوع، لطفاً به تصویر زیر نگاهی بیندازید. در اینجا، نحوه استفاده از دستور continue درون حلقه‌های do while، while. و for و نحوه عملکرد دقیق دستور continue را نشان می‌دهم.

![دستور Continue در زبان سی شارپ چگونه کار می‌کند؟](https://dotnettutorials.net/wp-content/uploads/2022/05/How-Does-the-Continue-Statement-Work-in-C-Language.jpg "دستور Continue در زبان سی شارپ چگونه کار می‌کند؟")

اگر به کد بالا توجه کرده باشید، ما دستور شرطی if را داخل بدنه حلقه نوشته‌ایم و در داخل بلوک شرط if، دستور continue را نوشته‌ایم. بنابراین، وقتی حلقه اجرا می‌شود، در هر تکرار، شرط if بررسی می‌شود و اگر شرط نادرست باشد، دستوراتی که بعد از بلوک if وجود دارند اجرا می‌شوند و تکرار بعدی ادامه می‌یابد. حال، وقتی شرط if درست باشد چه اتفاقی می‌افتد؟ به محض اینکه شرط if به true ارزیابی شود، بلوک if اجرا می‌شود و به محض اینکه دستور continue داخل بلوک if اجرا شود، از اجرای دستوراتی که بعد از دستور continue وجود دارند صرف نظر می‌کند و اجرای تکرار بعدی حلقه را ادامه می‌دهد.

##### **مثال برای درک دستور Continue در زبان سی شارپ:**

در مثال زیر، شرطی را برای اجرای حلقه ۵ بار، یعنی با شروع از مقدار I از ۱ تا ۵، تعیین کرده‌ایم. اما نیاز ما این است که وقتی مقدار I برابر با ۳ شد، باید از اجرای بدنه حلقه صرف نظر کنیم و تکرار بعدی را ادامه دهیم. در این حالت، باید شرط if را درون بدنه حلقه بنویسیم و بررسی کنیم که آیا مقدار I فعلی برابر با ۳ است یا خیر. اگر برابر با ۳ نباشد، اجرای حلقه for را ادامه داده و تکرار بعدی را اجرا می‌کنیم. اما اگر مقدار I برابر با ۳ باشد، شرط if مقدار true را برمی‌گرداند و در این صورت، دستور continue اجرا می‌شود و پس از اجرای دستور continue، بقیه دستور تکرار فعلی اجرا نمی‌شود و کنترل را به ادامه تکرار بعدی منتقل می‌کند.

```csharp
using System;

namespace JumpStatementDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            for (int i = 1; i <= 5; i++)
            {
                if (i == 3)
                {
                    continue;
                }
                Console.WriteLine($"I : {i}");
            }
            
            Console.ReadKey();
        }
    }
}
```

##### **استفاده از دستور Continue درون حلقه تو در تو:**

دستور continue یک تکرار جدید از نزدیکترین دستور تکرار محصور کننده (for، for each، while یا do loop) را آغاز می‌کند. برای درک بهتر، لطفاً به مثال زیر نگاهی بیندازید. در اینجا، حلقه بیرونی ۵ بار اجرا خواهد شد. مجدداً، برای هر تکرار حلقه بیرونی، شرط حلقه for داخلی را برای ۵ بار اجرا نوشته‌ایم. اما، در داخل بدنه حلقه داخلی، دستور continue را با استفاده از شرط if نوشته‌ایم و وقتی مقدار داخلی برابر با ۳ باشد، از اجرای بقیه دستورات صرف نظر می‌کند.

```csharp
using System;

namespace JumpStatementDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            for (int outer = 1; outer <= 5; outer++)
            {
                Console.WriteLine($"Outer: {outer}");
                for (int inner = 1; inner <= 5; inner++)
                {
                    if (inner == 3)
                    {
                        continue;
                    }
                    Console.Write($"  Inner: {inner}");
                }
                Console.WriteLine();
            }

            Console.ReadKey();
        }
    }
}
```

ما باید از استفاده از «دستور ادامه» در صورت امکان خودداری کنیم. دستور ادامه، برخی از دستورات حلقه را اجرا می‌کند و سپس بدون اجرای برخی از دستورات بعد از آن، از حلقه خارج می‌شود. می‌توانیم برای این منظور به جای ادامه، از دستور if استفاده کنیم. در مثال زیر، ما از دستور continue استفاده نمی‌کنیم و فقط با استفاده از شرط if به همان خروجی دست می‌یابیم.

```csharp
using System;

namespace JumpStatementDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            for (int outer = 1; outer <= 5; outer++)
            {
                Console.WriteLine($"Outer: {outer}");
                for (int inner = 1; inner <= 5; inner++)
                {
                    if (inner != 3)
                    {
                        Console.Write($"  Inner: {inner}");
                    }
                }
                Console.WriteLine();
            }

            Console.ReadKey();
        }
    }
}
```

**نکته:** وقتی نقطه‌ویرگول (;) بعد از while قرار می‌گیرد، حلقه به یک حلقه‌ی ساختگی تبدیل می‌شود. وقتی حلقه‌ی ساختگی ایجاد می‌شود، کامپایلر یک بدنه‌ی خالی بدون هیچ دستوری ایجاد می‌کند و بدنه‌ی فعلی به صورت outside درمی‌آید. سپس به طور خودکار continue در خارج از حلقه قرار می‌گیرد و در این صورت خطا رخ می‌دهد.
