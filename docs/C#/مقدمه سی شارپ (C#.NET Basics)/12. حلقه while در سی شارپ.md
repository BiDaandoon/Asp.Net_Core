
## **حلقه while در سی شارپ به همراه مثال**

در این مقاله، قصد دارم **حلقه While را در زبان C#** با مثال‌هایی مورد بحث قرار دهم. در پایان این مقاله، شما با مثال‌هایی در مورد دستورات حلقه‌ای و انواع آنها آشنا خواهید شد.

##### **حلقه while در زبان سی شارپ:**

یک حلقه چیزی نیست جز اجرای مکرر یک بلوک از دستورالعمل‌ها یا عبارات تا زمانی که شرط حلقه درست باشد. تعداد دفعاتی که تکرار می‌شود به معنای تا زمانی است که شرط داده شده درست باشد. هنگامی که شرط با شکست مواجه شود، اجرای حلقه خاتمه می‌یابد.

حلقه while برای اجرای مکرر یک دستور تا زمانی که شرط داده شده false را برگرداند، استفاده می‌شود. در اینجا، دستورات می‌توانند یک دستور واحد یا مجموعه‌ای از دستورات باشند. حلقه تا زمانی که شرط درست باشد، تکرار می‌شود. اگر به طور موازی به سینتکس و نمودار جریان نگاه کنید، درک بهتری از حلقه while خواهید داشت.

##### **ساختار حلقه while در زبان سی شارپ:**

در زیر سینتکس استفاده از حلقه while در زبان سی شارپ آمده است.

![سینتکس حلقه while در زبان سی شارپ](https://dotnettutorials.net/wp-content/uploads/2022/04/what-is-while-loop-in-c-language.png "سینتکس حلقه while در زبان سی شارپ")

در حالی که ابتدا با حلقه while کار می‌کنیم، باید شرط را بررسی کنیم، اگر شرط درست باشد، کنترل به داخل بدنه حلقه منتقل می‌شود و اگر شرط نادرست باشد، کنترل به خارج از بدنه حلقه منتقل می‌شود.

وقتی با یک دستور تکرار کار می‌کنیم، پس از اجرای بدنه، کنترل به شرط برگردانده می‌شود و تا زمانی که شرط نادرست شود، ادامه می‌یابد. اگر شرط نادرست نباشد، یک حلقه بی‌نهایت خواهیم داشت.

این چیزی شبیه به شرط if، شرط just و دستورات است، اما جریان آن با شرط if متفاوت است. اینکه چگونه متفاوت است، به ما اجازه می‌دهد تا از طریق نمودار جریان بفهمیم.

##### **نمودار جریان حلقه while در زبان سی شارپ:**

نمودار زیر نمودار جریان حلقه while را نشان می‌دهد.

![نمودار جریان حلقه while در زبان سی شارپ](https://dotnettutorials.net/wp-content/uploads/2022/04/flow-chart-of-while-loop-in-c-language.png "نمودار جریان حلقه while در زبان سی شارپ")

نمودار جریان شروع می‌شود. شروع با نماد بیضی نشان داده می‌شود. سپس شرط بررسی می‌شود. همانطور که قبلاً بحث شد، هر شرط دو خروجی دارد، یعنی درست و نادرست. اگر درست باشد چه اتفاقی خواهد افتاد و اگر نادرست باشد چه اتفاقی خواهد افتاد، باید بررسی کنیم.

فرض کنید شرط درست باشد، آنگاه تمام دستورات تعریف شده درون بلوک (درون بلوک حلقه while) اجرا خواهند شد. آیا پس از اجرای دستورات، حلقه به پایان می‌رسد؟ خیر، به پایان نمی‌رسد. پس از اجرای دستورات، حلقه دوباره به بررسی شرط می‌پردازد. تا زمانی که شرط داده شده درست باشد، همین فرآیند را تکرار می‌کند. فرض کنید شرط نادرست باشد، آنگاه حلقه به پایان می‌رسد. این جریان اجرای یک حلقه while است.

##### **مثال برای درک حلقه While در زبان سی شارپ:**

در مثال زیر، متغیر x با مقدار ۱ مقداردهی اولیه شده و سپس برای شرط بررسی شده است. اگر شرط مقدار درست را برگرداند، دستورات داخل بدنه حلقه while اجرا می‌شوند، در غیر این صورت کنترل از حلقه خارج می‌شود. مقدار **x** با استفاده از عملگر ++ افزایش می‌یابد، سپس دوباره برای شرط حلقه بررسی می‌شود.

```csharp
using System;

namespace ControlFlowDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            int x = 1;
            while (x <= 5)
            {
                Console.WriteLine("Value of x:" + x);
                x++;
            }
            Console.ReadKey();
        }
    }
}
```

##### **مثال: یک عدد وارد کنید و با استفاده از حلقه while در زبان سی شارپ، سری فیبوناچی را تا آن عدد چاپ کنید.**

```csharp
using System;

namespace ControlFlowDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            int i, n, j, k;
            Console.Write("Enter a Number : ");
            n = Convert.ToInt32(Console.ReadLine());
            i = 0;
            j = 1;
            Console.Write($"{i} {j}");
           
            k = i + j;
            while (k <= n)
            {
                Console.Write($" {k}");
                
                i = j;
                j = k;
                k = i + j;
            }

            Console.ReadKey();
        }
    }
}
```

##### **فرآیند پیش بررسی یا حلقه کنترل ورود چیست؟**

فرآیند پیش‌بررسی به این معنی است که قبل از ارزیابی بلوک دستور، بخش شرطی اجرا خواهد شد. وقتی با یک حلقه while کار می‌کنیم، همیشه فرآیند پیش‌بررسی رخ می‌دهد. حلقه‌ای که در آن قبل از اجرای بدنه حلقه، اگر ابتدا شرط بررسی شود، آن را حلقه کنترل‌شده با ورودی می‌نامند.

حلقه while نمونه‌ای از یک حلقه با ورودی کنترل‌شده است، زیرا در حلقه while قبل از اجرای بدنه، ابتدا شرط ارزیابی می‌شود، اگر شرط درست باشد، بدنه اجرا می‌شود، در غیر این صورت بدنه نادیده گرفته می‌شود.

##### **حلقه While تو در تو در زبان برنامه نویسی سی شارپ:**

نوشتن یک حلقه while درون حلقه while دیگر، حلقه while تودرتو نامیده می‌شود یا می‌توان گفت تعریف یک حلقه while درون حلقه while دیگر، حلقه while تودرتو نامیده می‌شود. به همین دلیل است که حلقه‌های تودرتو، «حلقه‌های درون حلقه» نیز نامیده می‌شوند. بسته به پیچیدگی مسئله داده شده، می‌توان هر تعداد حلقه را درون یکدیگر با هر یک از سه ترکیب موجود داشت. در پیاده‌سازی، وقتی نیاز داریم بدنه حلقه را n بار تکرار کنیم، باید به سراغ حلقه‌های تودرتو برویم.

##### **سینتکس حلقه While تو در تو در زبان سی شارپ:**

در زیر سینتکس استفاده از حلقه while تو در تو در زبان سی شارپ آمده است.

![سینتکس حلقه While تو در تو در زبان سی شارپ](https://dotnettutorials.net/wp-content/uploads/2022/04/nested-while-loop-in-c-programming-language.png "سینتکس حلقه While تو در تو در زبان سی شارپ")

**نکته:** در حلقه while تو در تو، تعداد تکرارها برابر با تعداد تکرارهای حلقه بیرونی ضربدر تعداد تکرارهای حلقه داخلی خواهد بود. حلقه‌های while تو در تو بیشتر برای ساخت برنامه‌های الگویی مختلف در سی شارپ مانند الگوهای عددی یا الگوهای شکلی استفاده می‌شوند.

##### **روند اجرای حلقه While تو در تو در زبان سی شارپ:**

حلقه while بیرونی بر اساس شرط بیرونی و حلقه while درونی بر اساس شرط درونی اجرا می‌شود. حال بیایید نحوه اجرای حلقه while تو در تو را بررسی کنیم. ابتدا، شرط حلقه بیرونی بررسی می‌شود و اگر شرط حلقه بیرونی برقرار نباشد، حلقه خاتمه می‌یابد.

فرض کنید شرط حلقه بیرونی درست باشد، سپس به داخل حلقه می‌آید، ابتدا دستورات حلقه بیرونی را که قبل از حلقه داخلی وجود دارند چاپ می‌کند. سپس شرط حلقه داخلی را بررسی می‌کند. اگر شرط while داخلی درست باشد، کنترل به داخل حلقه می‌رود و دستورات حلقه داخلی را اجرا می‌کند. پس از اجرای دستورات حلقه داخلی، دوباره شرط حلقه داخلی را بررسی می‌کند زیرا این یک حلقه است و تا زمانی که شرط درست باشد، این فرآیند تکرار می‌شود. به محض اینکه شرط حلقه داخلی ناموفق باشد، کنترل به خارج از حلقه می‌رود و دستوراتی را که بعد از حلقه داخلی وجود دارند اجرا می‌کند. پس از اجرا، دوباره به بررسی شرط حلقه بیرونی می‌پردازد. و اگر درست باشد، دوباره همان فرآیند را اجرا می‌کند. بنابراین، وقتی حلقه خاتمه می‌یابد به این معنی است که شرط حلقه بیرونی نادرست شود.

##### **نمودار جریان حلقه While تو در تو:**

لطفاً به نمودار زیر که نمودار جریان حلقه while تو در تو را نشان می‌دهد، نگاهی بیندازید.

![نمودار جریان حلقه While تو در تو](https://dotnettutorials.net/wp-content/uploads/2022/04/flow-chart-of-nested-while-loop.png "نمودار جریان حلقه While تو در تو")

جریان شروع می‌شود و ابتدا، شرط حلقه while بیرونی را بررسی می‌کند. و اگر شرط حلقه while بیرونی ناموفق باشد، به پایان می‌رسد. فرض کنید شرط حلقه while بیرونی درست باشد، سپس ابتدا دستورات حلقه while بیرونی را در صورت وجود اجرا می‌کند. پس از اجرای دستورات حلقه while بیرونی، شرط حلقه while درونی را بررسی می‌کند. برای شرط حلقه while درونی نیز، درست و نادرست بودن را بررسی می‌کند. فرض کنید شرط حلقه while درونی درست باشد، سپس دستورات حلقه while درونی اجرا می‌شوند. پس از اجرای دستورات حلقه while درونی، دوباره شرط حلقه while درونی بررسی می‌شود و این فرآیند اجرای حلقه درونی تا زمانی که شرط حلقه while درونی درست باشد، تکرار می‌شود. اگر شرط حلقه while درونی نادرست باشد، دستورات حلقه بیرونی باقی مانده اجرا می‌شوند. پس از اجرای دستورات حلقه while بیرونی، دوباره دستورات حلقه while بیرونی بررسی می‌شوند. این جریان حلقه while تو در تو است.

##### **مثال برای چاپ فرمت زیر با استفاده از حلقه While تو در تو در زبان سی شارپ**

![مثال برای چاپ فرمت زیر با استفاده از حلقه While تو در تو در زبان سی شارپ](https://dotnettutorials.net/wp-content/uploads/2022/04/nested-while-loop-in-c-programming-language-with-e.png "مثال برای چاپ فرمت زیر با استفاده از حلقه While تو در تو در زبان سی شارپ")

###### **برنامه:**

```csharp
using System;

namespace ControlFlowDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.Write("ENTER  A NUMBER ");
            int n = Convert.ToInt32(Console.ReadLine());
            int i = 1;
            while (i <= n)
            {
                Console.WriteLine();
                int j = 1;
                while (j <= i)
                {
                    Console.Write(j + " ");
                    j++;
                }
                i++;
            }

            Console.ReadKey();
        }
    }
}
```
