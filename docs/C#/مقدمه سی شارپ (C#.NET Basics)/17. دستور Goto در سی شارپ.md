## **دستور Goto در سی شارپ به همراه مثال**

در این مقاله، قصد دارم **دستور Goto در سی شارپ را** با مثال‌هایی مورد بحث قرار دهم. در پایان این مقاله، شما دستور goto در سی شارپ و زمان و نحوه استفاده از دستور goto در زبان سی شارپ را با مثال‌هایی خواهید فهمید.

##### **دستور Goto در سی شارپ**

دستور Goto در سی شارپ برای انتقال کنترل به دستور برچسب گذاری شده در برنامه استفاده می‌شود. برچسب یک شناسه معتبر است و درست قبل از دستوری که کنترل از آنجا منتقل می‌شود قرار می‌گیرد. این بدان معناست که دستور goto یک پرش بدون قید و شرط از goto به یک دستور برچسب گذاری شده در همان تابع فراهم می‌کند.

Goto یک کلمه کلیدی است و با استفاده از این کلمه کلیدی goto می‌توانیم کنترل را به هر کجای برنامه در محدوده محلی منتقل کنیم. وقتی با دستور goto کار می‌کنیم، به یک شناسه به نام label نیاز دارد. هر شناسه معتبری که به دنبال آن یک دونقطه بیاید، label نامیده می‌شود. هر زمان که با دستور goto کار می‌کنیم، به آن یک دستور جریان کنترل بدون ساختار گفته می‌شود زیرا قانون زبان برنامه‌نویسی ساختار یافته را می‌شکند.

دستور goto به ندرت استفاده می‌شود زیرا برنامه را گیج‌کننده، ناخوانا و پیچیده می‌کند. همچنین، وقتی از آن استفاده می‌شود، کنترل برنامه به راحتی قابل ردیابی نخواهد بود، از این رو تست و اشکال‌زدایی را دشوار می‌کند.

##### **مثال برای درک دستور goto در زبان سی شارپ:**

بیایید با یک مثال دستور goto را در سی شارپ درک کنیم. لطفاً به کد زیر نگاهی بیندازید. در اینجا، می‌توانید ببینید که ما یک برچسب (یا شناسه) به نام Label1 ایجاد کرده‌ایم و درون متد Main پس از چاپ عدد 1، در این حالت، کنترل مستقیماً به برچسب Label1 پرش می‌کند و اجرا را از دستور سوم شروع می‌کند. در این مورد، دستور دوم اجرا نخواهد شد.

```csharp
using System;

namespace JumpStatementDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("1st Statement: ");

            goto label1;   //goto label label1:

            Console.WriteLine("2nd Statement: ");

        label1:    //label:
            Console.WriteLine("3rd Statement: ");

            Console.WriteLine("End of Main Method Statement: ");
            Console.ReadKey();
        }
    }
}
```

##### **استفاده از دستور goto درون حلقه در سی شارپ:**

دستور goto کنترل برنامه را به یک دستور برچسب‌گذاری شده منتقل می‌کند. دستور label باید در محدوده دستور goto وجود داشته باشد. بیش از یک دستور goto می‌توانند کنترل را به یک برچسب منتقل کنند. این دستور می‌تواند برای خروج از یک حلقه یا یک حلقه تو در تو داخلی به یک حلقه خارجی استفاده شود. برای درک بهتر، لطفاً به مثال زیر نگاهی بیندازید. در مثال زیر، ما با استفاده از دستور goto از حلقه خارج می‌شویم. در این حالت، دستوری که بعد از بدنه حلقه و قبل از دستور label نوشته شده است، اجرا نخواهد شد.

```csharp
using System;

namespace JumpStatementDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            for (int i = 1; i <= 10; i++)
            {
                if (i == 5)
                {
                    goto Exitlabel;
                }
                Console.WriteLine(i);
            }
            Console.WriteLine("Next statement placed after loop");

            Exitlabel: //goto label
                Console.WriteLine("Labeled statement");

            Console.ReadKey();
        }
    }
}
```

##### **استفاده از goto در دستور Switch**

برخلاف دستور break، این دستور کنترل برنامه را به دستور بعدی که بلافاصله پس از حلقه یا switch قرار می‌گیرد، منتقل نمی‌کند، بلکه کنترل را به دستوری که پس از برچسب منطبق نوشته شده است، منتقل می‌کند.

همچنین می‌توانید از دستور goto برای انتقال کنترل به یک برچسب switch-case خاص یا برچسب پیش‌فرض در یک دستور switch استفاده کنید. برای درک بهتر، لطفاً به مثال زیر نگاهی بیندازید. در مثال زیر، در case 20، به جای break، goto case 5 را نوشته‌ایم. بنابراین، در این حالت، کنترل به case 5 منتقل می‌شود.

```csharp
using System;

namespace JumpStatementDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            int number = 20;
            switch (number)
            {

                case 5:
                    Console.WriteLine("case 5");
                    break;
                case 10:
                    Console.WriteLine("case 10");
                    break;
                case 20:
                    Console.WriteLine("case 20");
                    // goto statement transfer
                    // the control to case 5
                    goto case 5;

                default:
                    Console.WriteLine("No match found");
                    break;
            }
            Console.ReadKey();
        }
    }
}
```

##### **آیا می‌توانیم اعداد ۱ تا ۱۰ را بدون استفاده از حلقه در سی شارپ چاپ کنیم؟**

این یکی از سوالات متداول مصاحبه کتبی است. مصاحبه‌کننده از شما می‌خواهد برنامه‌ای بنویسید که اعداد ۱ تا n را بدون استفاده از حلقه چاپ کند. در سی‌شارپ این امکان وجود دارد که بدون استفاده از حلقه نیز بتوانیم با استفاده از دستور goto به قابلیت حلقه دست یابیم. برای درک بهتر، لطفاً به مثال زیر نگاهی بیندازید. در مثال زیر، ما اعداد ۱ تا ۱۰ را با استفاده از دستور goto چاپ می‌کنیم. در این حالت، هر بار مقدار متغیر count بررسی می‌شود و اگر کمتر یا مساوی ۱۰ باشد، دستور goto اجرا می‌شود. وقتی مقدار count برابر با ۱۱ شود، شرط if نادرست می‌شود و دستور goto اجرا نخواهد شد.

```csharp
using System;

namespace ControlFlowDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            int count = 1;
        label_Loop:
            Console.WriteLine(count);
            count++;
            if (count <= 10)
            {
                goto label_Loop;
            }

            Console.ReadKey();
        }
    }
}
```

**نکته:** ما باید تا حد امکان از استفاده از «دستور goto» خودداری کنیم. ردیابی نحوه اجرا و فهمیدن اینکه برنامه چه کاری انجام می‌دهد بسیار دشوار است و اشکال‌زدایی و اصلاح چنین برنامه‌ای بسیار دشوار است. و تمام برنامه‌های ما باید شامل توالی‌ها، تصمیمات و حلقه‌ها باشند.
