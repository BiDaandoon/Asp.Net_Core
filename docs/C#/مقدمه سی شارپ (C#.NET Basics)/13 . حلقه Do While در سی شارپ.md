## **حلقه Do While در سی شارپ با مثال انجام دهید**

در این مقاله، قصد دارم **حلقه Do While در سی شارپ را** با مثال‌هایی مورد بحث قرار دهم. در پایان این مقاله، شما خواهید فهمید که حلقه do-while چیست و چه زمانی و چگونه می‌توان از یک حلقه do-while در زبان سی شارپ با مثال‌ها استفاده کرد.

##### **حلقه Do While در زبان سی شارپ اجرا کنید:**

حلقه do-while یک حلقه پس از آزمایش یا حلقه کنترل خروج است، یعنی ابتدا بدنه حلقه را اجرا می‌کند و سپس شرط را آزمایش می‌کند. این بدان معناست که ما باید از حلقه do-while استفاده کنیم که در آن باید حداقل یک بار بدنه حلقه را اجرا کنیم. حلقه do-while عمدتاً در برنامه‌های مبتنی بر منو استفاده می‌شود که در آن شرط خاتمه به کاربر نهایی بستگی دارد. این بدان معناست که وقتی کاربر نهایی بخواهد، حلقه خاتمه می‌یابد. اگر در حال حاضر این موضوع واضح نیست، نگران نباشید، این را با یک مثال برنامه مبتنی بر منو درک کنید.

##### **سینتکس استفاده از حلقه Do While در زبان سی شارپ:**

تصویر زیر سینتکس استفاده از حلقه do while را در زبان سی شارپ نشان می‌دهد. این حلقه با استفاده از کلمه کلیدی do و به دنبال آن باز و بسته کردن آکولاد ایجاد می‌شود. بین باز و بسته کردن آکولاد، می‌توانید دستوراتی را که می‌خواهید حداقل یک بار اجرا شوند، بنویسید. و بعد از بستن آکولاد، باید شرط while را بنویسید. لطفاً توجه داشته باشید که عبارت شرط while با یک نقطه ویرگول به پایان می‌رسد. عبارت شرط باید یک عبارت بولی باشد.

##### **دستورهای while و do-while تقریباً یکسان هستند. خب، تفاوتشان چیست؟ از کدام یک استفاده کنیم؟**

تفاوت بین حلقه do-while و حلقه while در سی شارپ این است که do-while شرط تست خود را در پایین حلقه ارزیابی می‌کند در حالی که حلقه while شرط تست خود را در بالای حلقه ارزیابی می‌کند. بنابراین، دستورات نوشته شده در داخل بلوک do-while حداقل یک بار اجرا می‌شوند در حالی که ما نمی‌توانیم تضمین کنیم که دستورات نوشته شده در داخل حلقه while حداقل یک بار اجرا شوند.

**نکته:** وقتی می‌خواهید بدنه حلقه را حداقل یک بار، صرف نظر از شرط، اجرا کنید، باید از حلقه do-while استفاده کنید، در غیر این صورت باید از حلقه while استفاده کنید.

##### **نمودار جریان حلقه Do-While:**

در حلقه do-while، ابتدا بدنه حلقه بدون بررسی شرط اجرا می‌شود. پس از اجرای بدنه حلقه، شرط بررسی می‌شود و اگر شرط درست باشد، دوباره بدنه حلقه اجرا می‌شود و اگر شرط نادرست باشد، حلقه خاتمه می‌یابد. برای درک بهتر، لطفاً به نمودار زیر که فلوچارت حلقه do-while را نشان می‌دهد، نگاهی بیندازید.

##### **مثال برای درک حلقه do while در زبان سی شارپ:**

در مثال زیر ما اعداد ۱ تا ۵ را با استفاده از حلقه do while چاپ می‌کنیم.

```csharp
using System;

namespace ControlFlowDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            int number = 1;
            do
            {
                Console.Write($"{number} ");
                number++;
            } while (number <= 5);

            Console.ReadKey();
        }
    }
}
```

```csharp
using System;

namespace ControlFlowDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            do
            {
                Console.WriteLine("I am from outer do-while loop");
                do
                {
                    Console.WriteLine("I am from inner do-while loop ");
                }
                while (1 > 10);
            }
            while (2 > 10);

            Console.ReadKey();
        }
    }
}
```

##### **مثال بلادرنگ از حلقه Do-While در سی شارپ:**

بیایید حلقه do-while را با یک مثال بلادرنگ درک کنیم. حلقه do-while عمدتاً در برنامه‌های مبتنی بر منو استفاده می‌شود که در آن‌ها شرط خاتمه به کاربر نهایی بستگی دارد. این بدان معناست که وقتی کاربر نهایی بخواهد، حلقه خاتمه می‌یابد.

برای درک بهتر، لطفاً به مثال زیر نگاهی بیندازید. در مثال زیر، ما از یک حلقه do while استفاده می‌کنیم و درون حلقه do while از یک دستور switch case استفاده می‌کنیم. در اینجا، ابتدا از کاربر می‌خواهیم که گزینه‌ها را انتخاب کند، یعنی عملیاتی که می‌خواهد انجام دهد. گزینه‌ها ۱ برای جمع، ۲ برای تفریق، ۳ برای ضرب و ۴ برای تقسیم هستند. بر اساس گزینه‌های انتخاب شده، ما دستور matched case را اجرا می‌کنیم و عملیات مورد نیاز را انجام می‌دهیم. پس از انجام عملیات، حلقه را خاتمه نمی‌دهیم، بلکه از کاربر می‌پرسیم که آیا می‌خواهد ادامه دهد یا می‌خواهد خاتمه دهد. اگر حرف y را وارد کند، دوباره گزینه‌هایی را برای انتخاب و انجام عملیات مورد نظر نشان می‌دهیم و اگر هر چیزی غیر از y وارد کند، حلقه خاتمه می‌یابد و برنامه به پایان می‌رسد.

```csharp
using System;

namespace ControlFlowDemo
{
    class Program
    {
        static void Main(string[] args)
        {
            char Choice;
            int MenuOption;
            int Number1, Number2;
            do
            {
                Console.WriteLine("Press 1 for Addition, 2 for Subtraction, 3 for Multiplication, 4 for Division");
                MenuOption = Convert.ToInt32(Console.ReadLine());
                switch (MenuOption)
                {
                    case 1:
                        Console.WriteLine("Enter the value of two numbers");
                        Number1 = Convert.ToInt32(Console.ReadLine());
                        Number2 = Convert.ToInt32(Console.ReadLine());
                        Console.WriteLine($"Sum Is {Number1 + Number2}");
                        break;
                    case 2:
                        Console.WriteLine("Enter the value of two numbers");
                        Number1 = Convert.ToInt32(Console.ReadLine());
                        Number2 = Convert.ToInt32(Console.ReadLine());
                        Console.WriteLine($"Difference Is {Number1 - Number2}");
                        break;
                    case 3:
                        Console.WriteLine("Enter the value of two numbers");
                        Number1 = Convert.ToInt32(Console.ReadLine());
                        Number2 = Convert.ToInt32(Console.ReadLine());
                        Console.WriteLine($"Multiplication Is {Number1 * Number2}");
                        break;
                    case 4:
                        Console.WriteLine("Enter the value of two numbers");
                        Number1 = Convert.ToInt32(Console.ReadLine());
                        Number2 = Convert.ToInt32(Console.ReadLine());
                        Console.WriteLine($"Division Is {Number1 / Number2}");
                        break;
                    default:
                        Console.WriteLine("Invalid choice");
                        break;
                }
                Console.WriteLine("Please Enter Y to continue, any keys to terminate");

                Choice = Convert.ToChar(Console.ReadLine());
            }
            while (Char.ToUpper(Choice) == 'Y');
        }
    }
}
```
