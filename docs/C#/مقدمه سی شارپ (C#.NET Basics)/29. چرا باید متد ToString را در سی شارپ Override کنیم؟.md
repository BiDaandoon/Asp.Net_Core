## **چرا باید متد ToString را در سی شارپ Override کنیم؟**

##### **آشنایی با کلاس شیء در سی شارپ:**

کلاس Object، سوپرکلاس تمام انواع dot net است. به این معنی که تمام انواع در .NET Framework به طور مستقیم یا غیرمستقیم از کلاس Object به ارث می‌رسند. به دلیل این ارث‌بری، هر نوع در .NET متد ToString() را از کلاس Object به ارث می‌برد. اگر به تعریف کلاس Object بروید، خواهید دید که متد ToString() به عنوان یک متد مجازی تعریف شده است که اجازه می‌دهد این متد در کلاس‌های فرزند بازنویسی شود. نه تنها متد ToString، بلکه می‌توانید متد Equals را نیز بازنویسی کنید که در مقاله بعدی به آن خواهیم پرداخت.

![کلاس شیء در سی شارپ](https://dotnettutorials.net/wp-content/uploads/2018/09/Object-Class-in-C.jpg "کلاس شیء در سی شارپ")

**نکته:** هر نوع داده در .NET به طور ضمنی از کلاس Object ارث‌بری می‌شود و از این رو تمام اعضای public و protected کلاس object (به استثنای اعضای private) به کلاس فرزند ارث‌بری می‌شوند و با استفاده از object کلاس فرزند می‌توانیم به تمام اعضای public و protected کلاس object در C# دسترسی داشته باشیم. به عنوان مثال، int یک نوع داده اولیه و string یک نوع داده مرجع است و هر دوی این دو نوع داده از کلاس Object ارث‌بری می‌شوند و از این رو با استفاده از متغیر int و string می‌توانیم به تمام اعضای public و protected کلاس object دسترسی داشته باشیم.

به عبارت دیگر، می‌توانیم بگوییم که هر نوع کلاس (انواع مرجع) یا نوع ساختار (انواع مقداری) به طور مستقیم یا غیرمستقیم و ضمنی از کلاس Object در C# به ارث می‌رسند. بنابراین، هر شیء در C# متد ToString را دریافت می‌کند که یک نمایش رشته‌ای از آن شیء را برمی‌گرداند. بنابراین، متد ToString() رشته‌ای را برمی‌گرداند که نشان دهنده شیء فعلی است.

برای مثال، تمام متغیرهای نوع int یا float دارای متد ToString هستند که به آنها امکان می‌دهد محتویات خود را به صورت رشته برگردانند. برای درک بهتر، لطفاً به مثال زیر نگاهی بیندازید. در مثال بالا، متغیر Number یک متغیر از نوع عدد صحیح است و وقتی متد ToString() را روی شیء Number فراخوانی می‌کنیم، نمایش رشته‌ای از عدد صحیح ۱۰۰ را به ما می‌دهد.

```csharp
using System;

namespace UnderstandingObjectClassMethods
{
    public class Program
    {
        public static void Main()
        {
            int Number = 100;
            Console.WriteLine(Number.ToString());
        }
    }
}
```

وقتی یک کلاس یا ساختار سفارشی در سی شارپ ایجاد می‌کنید، می‌توانید متد ToString را برای ارائه اطلاعات در مورد نوع داده خود به کلاینت، بازنویسی کنید. برای مثال، اگر یک نوع داده پیچیده دارید، مثلاً کلاس Employee، همانطور که در مثال زیر نشان داده شده است، و وقتی متد ToString() را روی شیء Employee فراخوانی می‌کنید، خروجی مورد انتظار را دریافت نخواهید کرد. از این رو، باید متد ToString() را که از کلاس Object به ارث رسیده است، بازنویسی کنیم .

```csharp
using System;

namespace UnderstandingObjectClassMethods
{
    public class Program
    {
        public static void Main()
        {
            Employee emp = new Employee();
            emp.FirstName = "Pranaya";
            emp.LastName = "Rout";
            Console.WriteLine(emp.ToString());
            Console.ReadKey();
        }
    }

    public class Employee
    {
        public string FirstName;
        public string LastName;
    }
}
```

وقتی کد بالا را اجرا می‌کنید، خروجی زیر را به شما می‌دهد. این کد نام کامل نوع Employee را به ما می‌دهد.

![چرا باید متد ToString را در سی شارپ override کنیم؟](https://dotnettutorials.net/wp-content/uploads/2018/09/word-image-144.png "چرا باید متد ToString را در سی شارپ override کنیم؟")

نیاز ما این است که وقتی متد ToString() را فراخوانی می‌کنیم، نام و نام خانوادگی شیء Employee نمایش داده شود. برای رسیدن به این هدف، باید متد مجازی ToString() را که توسط کلاس Object در C# ارائه می‌شود، بازنویسی کنیم.

##### **بازنویسی متد ToString() در سی شارپ:**

لطفاً کد را مطابق شکل زیر تغییر دهید تا متد ToString() درون کلاس Employee را بازنویسی کنید. نکته‌ای که باید به خاطر داشته باشید این است که متد ToString به عنوان یک متد مجازی درون کلاس Object تعریف شده است و کلاس Employee سفارشی ما به طور ضمنی از کلاس Object به ارث رسیده است و از این رو درون این کلاس Employee، باید متد ToString را با استفاده از اصلاح‌کننده override که در مثال زیر نشان داده شده است، بازنویسی کنیم.

```csharp
using System;

namespace UnderstandingObjectClassMethods
{
    public class Program
    {
        public static void Main()
        {
            Employee emp = new Employee();
            emp.FirstName = "Pranaya";
            emp.LastName = "Rout";
            Console.WriteLine(emp.ToString());
            Console.ReadKey();
        }
    }

    public class Employee
    {
        public string FirstName;
        public string LastName;
        //Overriding the Virtual ToString method of Object class
        //Overriding the Virtual method using override modifier
        public override string ToString()
        {
            return FirstName + ", " + LastName;
        }
    }
}
```

اکنون برنامه را اجرا کنید و همانطور که در زیر نشان داده شده است، نام و نام خانوادگی کارمند را مشاهده خواهید کرد.
