##### **متد ناشناس در سی شارپ چیست؟**

همانطور که از نامش پیداست، یک متد ناشناس در سی شارپ، متدی بدون نام است. یا می‌توان گفت یک بلوک کد بدون نام. **متدهای ناشناس در سی شارپ با استفاده از کلمه کلیدی delegate تعریف می‌شوند و می‌توانند به متغیری از نوع delegate اختصاص داده شوند** . اگر در حال حاضر این موضوع برایتان واضح نیست، نگران نباشید، ما این موضوع را با مثال‌های متعدد درک خواهیم کرد.

##### **چرا به متدهای ناشناس در سی شارپ نیاز داریم؟**

در   [**مقاله‌ی Delegates در سی‌شارپ**](https://dotnettutorials.net/lesson/delegates-csharp/)   ، در مورد نحوه‌ی اتصال یک delegate به یک متد بحث کردیم. برای اتصال یک delegate به یک متد، ابتدا باید یک نمونه از delegate ایجاد کنیم و وقتی نمونه‌ی delegate را ایجاد می‌کنیم، باید نام متد را به عنوان پارامتر به سازنده‌ی delegate ارسال کنیم، و این تابعی است که delegate به آن اشاره می‌کند و وقتی delegate را فراخوانی می‌کنیم، این تابع که توسط delegate به آن اشاره شده است، اجرا خواهد شد. همچنین ممکن است delegateها به چندین تابع اشاره داشته باشند و در این صورت، وقتی نمونه‌ی delegate را فراخوانی می‌کنیم، تمام توابعی که توسط delegate به آنها اشاره شده است، اجرا می‌شوند.

##### **مثال برای درک Delegate در سی شارپ:**

قبل از درک متد ناشناس، ابتدا اجازه دهید ببینیم چگونه می‌توانیم از یک delegate برای اجرای متد استفاده کنیم. چرا؟ زیرا برای درک متد ناشناس، درک delegate مهم است زیرا متدهای ناشناس از delegate استفاده می‌کنند. مثال زیر نحوه اعلام یک delegate، نحوه ایجاد نمونه‌ای از یک delegate و نحوه فراخوانی delegate را نشان می‌دهد.

```csharp
using System;

namespace DelegateDemo
{
    public class AnonymousMethods
    {
        public delegate string GreetingsDelegate(string name);
        public static string Greetings(string name)
        {
            return "Hello @" + name + " Welcome to Dotnet Tutorials";
        }

        static void Main(string[] args)
        {
            GreetingsDelegate gd = new GreetingsDelegate(AnonymousMethods.Greetings);
            string GreetingsMessage = gd.Invoke("Pranaya");
            Console.WriteLine(GreetingsMessage);
            Console.ReadKey();
        }
    }
}
```

وقتی کد بالا را اجرا کنید، خروجی زیر را دریافت خواهید کرد.

![مثالی برای درک Delegate در سی شارپ](https://dotnettutorials.net/wp-content/uploads/2022/09/word-image-29911-1-10.png "مثالی برای درک Delegate در سی شارپ")

در مثال بالا،

1. ما یک نماینده (GreetingsDelegate) ایجاد می‌کنیم.
2. سپس نماینده را نمونه‌سازی می‌کنیم، در حین نمونه‌سازی نماینده، نام متد ( **Greetings** ) را به عنوان پارامتر به سازنده نماینده ارسال می‌کنیم.
3. در نهایت، delegate را فراخوانی می‌کنیم.

در حال حاضر، این رویکردی است که ما برای اتصال یک متد به یک نماینده سفارشی و اجرای آن دنبال می‌کنیم. یک متد ناشناس در سی شارپ نیز به یک نماینده مرتبط است. بدون اتصال یک بلوک نامگذاری شده (تابع یا متد) به یک نماینده، می‌توانیم یک بلوک کد (بلوک کد بدون نام) را نیز به یک نماینده متصل کنیم که چیزی جز یک متد ناشناس در سی شارپ نیست.

```csharp
            Console.ReadKey();

```

##### **دسترسی متد ناشناس به متغیرهای تعریف شده در خارج از کلاس:**

اول از همه، آیا می‌توانیم به متغیر درون یک متد ناشناس که خارج از متد ناشناس تعریف شده است، دسترسی پیدا کنیم؟ بله، می‌توانیم به آن دسترسی پیدا کنیم. لطفاً برای درک بهتر به مثال زیر نگاهی بیندازید. در مثال زیر، می‌توانید ببینید که ما به متغیر Message درون متد ناشناس که خارج از متد ناشناس تعریف شده است، دسترسی پیدا می‌کنیم.

```csharp
using System;

namespace DelegateDemo
{

    {
        public delegate string GreetingsDelegate(string name);

        static void Main(string[] args)
        {

            GreetingsDelegate gd = delegate (string name)
            {
                return "Hello @" + name + " " + Message;
            };

            string GreetingsMessage = gd.Invoke("Pranaya");
            Console.WriteLine(GreetingsMessage);
            Console.ReadKey();
        }
    }
}
```

اکنون، ما کلاس Employee را درون فایل کلاس Employee.cs ایجاد کرده‌ایم. حال، فایل کلاس Program.cs را به صورت زیر تغییر دهید تا مثال مورد بحث در بالا پیاده‌سازی شود.

```csharp
using System;
using System.Collections.Generic;

namespace AnonymousMethodRealTimeExample
{
    class Program
    {
        static void Main(string[] args)
        {
            // Step 3: 
            // Create an instance of Predicate\<Employee> delegate and 
            // pass the method name as an argument to the delegate constructor          
            Predicate<Employee> employeePredicate = new Predicate<Employee>(IsEmployeeExist);

            //Step4
            //Create a collection of List of Employees
            List<Employee> listEmployees = new List<Employee>()
            {
                new Employee{ ID = 101, Name = "Pranaya", Gender = "Male", Salary = 100000},
                new Employee{ ID = 102, Name = "Priyanka", Gender = "Female", Salary = 200000},
                new Employee{ ID = 103, Name = "Anurag", Gender = "Male", Salary = 300000},
                new Employee{ ID = 104, Name = "Preety", Gender = "Female", Salary = 400000},
                new Employee{ ID = 104, Name = "Sambit", Gender = "Male", Salary = 500000},
            };

            // Step 5: 
            // Now pass the delegate instance as the
            // argument to the Find() method of List collection
            Employee employee = listEmployees.Find(x => employeePredicate(x));
            Console.WriteLine(@"ID : {0}, Name : {1}, Gender : {2}, Salary : {3}",
                employee.ID, employee.Name, employee.Gender, employee.Salary);

            Console.ReadKey();
        }

        // Step 2: 
        // Create a method whose signature matches with the
        // signature of Predicate\<Employee> generic delegate
        public static bool IsEmployeeExist(Employee Emp)
        {
            return Emp.ID == 103;
        }
    }
}
```

3. سپس آن نمونه Predicate Generic Delegate را به عنوان آرگومان به متد Find از کلاس مجموعه List ارسال می‌کنیم.

با استفاده از متد ناشناس، می‌توانیم با خیال راحت از سه مرحله‌ی بالا اجتناب کنیم. می‌توانیم یک متد ناشناس را به عنوان آرگومان به متد Find() ارسال کنیم، همانطور که در کد زیر نشان داده شده است.

![استفاده از متد ناشناس در سی شارپ](https://dotnettutorials.net/wp-content/uploads/2022/09/word-image-29911-11-3.png "استفاده از متد ناشناس در سی شارپ")

با متد ناشناس بالا، می‌توانید به سادگی مرحله و مرحله ۲ را حذف کنید و مرحله ۵ را با کد بالا جایگزین کنید. بنابراین، با اعمال تغییرات بالا، اکنون کد کلاس Program باید به شکل زیر باشد.

```csharp
using System;
using System.Collections.Generic;

namespace AnonymousMethodRealTimeExample
{
    class Program
    {
        static void Main(string[] args)
        {
            //Step2
            //Create a collection of List of Employees
            List<Employee> listEmployees = new List<Employee>()
            {
                new Employee{ ID = 101, Name = "Pranaya", Gender = "Male", Salary = 100000},
                new Employee{ ID = 102, Name = "Priyanka", Gender = "Female", Salary = 200000},
                new Employee{ ID = 103, Name = "Anurag", Gender = "Male", Salary = 300000},
                new Employee{ ID = 104, Name = "Preety", Gender = "Female", Salary = 400000},
                new Employee{ ID = 104, Name = "Sambit", Gender = "Male", Salary = 500000},
            };

            // Step3
            // An anonymous method is being passed as an argument to
            // the Find() method of List Collection.
            Employee employee = listEmployees.Find(
                                    delegate (Employee x)
                                    {
                                        return x.ID == 103;
                                    }
                                );
            Console.WriteLine(@"ID : {0}, Name : {1}, Gender : {2}, Salary : {3}",
                employee.ID, employee.Name, employee.Gender, employee.Salary);

            Console.ReadKey();
        }
    }
}
```

##### **روش پیدا کردن در سی شارپ:**

در دو مثال بالا، **متد Find** () از کلاس مجموعه عمومی List انتظار دارد که یک delegate به عنوان آرگومان ارسال شود. اگر می‌خواهید امضای متد Find را ببینید، روی **متد Find()** کلیک راست کرده و **از منوی زمینه، گزینه Go To Definition** را انتخاب کنید . سپس امضای متد زیر از متد Find را مشاهده خواهید کرد. می‌توانید ببینید که یک پارامتر از نوع Generic Predicate دریافت می‌کند.
