## **مثال‌های رابط کاربری بلادرنگ در سی شارپ**

در این مقاله، قصد دارم در مورد **مثال‌های بلادرنگ رابط کاربری در سی‌شارپ** صحبت کنم ، یعنی نحوه استفاده از رابط کاربری در توسعه برنامه‌های سی‌شارپ با مثال‌های بلادرنگ. ل
##### **رابط (Interface) در سی شارپ چیست؟**

رابط در سی شارپ یک کلاس کاملاً پیاده‌سازی نشده است که برای اعلان مجموعه‌ای از متدهای یک شیء استفاده می‌شود. بنابراین، می‌توانیم یک رابط را به عنوان یک کلاس انتزاعی خالص تعریف کنیم که به ما امکان می‌دهد فقط متدهای انتزاعی را تعریف کنیم. متد انتزاعی به معنای متدی بدون بدنه یا پیاده‌سازی است.

پیاده‌سازی آن باید توسط کلاس یا ساختار ارائه شود. کلاس یا ساختاری که رابط را پیاده‌سازی می‌کند، باید پیاده‌سازی تمام متدهای تعریف‌شده درون رابط را نیز فراهم کند.

##### **مثال بلادرنگ از رابط در سی شارپ:**

اگر در مصاحبه‌ای شرکت می‌کنید، مصاحبه‌کننده ممکن است این سوال را از شما بپرسد که کاربرد رابط کاربری در برنامه‌های بلادرنگ چیست یا ممکن است از شما پرسیده شود که در کجای پروژه خود از رابط کاربری استفاده کرده‌اید؟

بگذارید با یک مثال واقعی، پاسخ را درک کنیم. فرض کنید قرار است یک برنامه برای یک بانک توسعه دهید. و ما امروزه می‌دانیم که می‌توانیم انواع مختلفی از حساب‌های بانکی، مثلاً حساب‌های پس‌انداز، حساب‌های جاری، حساب‌های Demat و غیره را باز کنیم. و ما از قبل ویژگی‌ها و خدمات همه این انواع مختلف حساب‌های بانکی را می‌دانیم.

حال، سوال این است که چگونه می‌توانیم این برنامه را پیاده‌سازی کنیم. بنابراین، ما قصد داریم یک برنامه بانکی توسعه دهیم که در آن به کاربر اجازه می‌دهیم حساب‌های زیر را ایجاد کند.

1. **حساب پس‌انداز**
2. **حساب جاری**

و در هر حساب، قرار است خدمات زیر را ارائه دهیم.

1. **مبلغ واریزی**
2. **مبلغ برداشت**
3. **چک‌بالنس**

بنابراین، چگونه می‌توانیم روش‌های فوق را پیاده‌سازی کنیم؟ ما یک رابط کاربری ایجاد خواهیم کرد، مثلاً IBankAccount را با تعریف سه سرویس فوق به شرح زیر:

```csharp
namespace BankApplication
{
    interface IBankAccount
    {
        bool DepositAmount(decimal amount);
        bool WithdrawAmount(decimal amount);
        decimal CheckBalance();
    }
}
```

در مرحله بعد، باید دو کلاس مختلف برای Saving Account، Current Account و Demat Account ایجاد کنیم و رابط IBankAccount فوق را پیاده‌سازی کرده و پیاده‌سازی متدهای رابط فوق را فراهم کنیم.

##### **حساب پس‌انداز:**

یک فایل کلاس با نام SavingAccount.cs ایجاد کنید و سپس کد زیر را کپی و جایگذاری کنید. در اینجا، می‌توانید ببینید که ما سه متغیر خصوصی ایجاد می‌کنیم. متغیر Balance برای نگهداری موجودی حساب کاربر استفاده می‌شود، PerDayWithdrawLimit برای محدود کردن برداشت کاربر استفاده می‌شود، یعنی ما 10000 در روز تنظیم می‌کنیم و متغیر TodayWithdrawal برای نگهداری مبلغ برداشت روز جاری استفاده می‌شود. کلاس ما رابط IBankAccount را پیاده‌سازی می‌کند و پیاده‌سازی هر سه متد انتزاعی را که در داخل رابط IBankAccount تعریف شده‌اند، فراهم می‌کند.

1. **DepositAmount:** این متد مبلغ واریزی را دریافت می‌کند و منطق آن بسیار سرراست است. هر مبلغی که دریافت کنیم، فقط آن را با متغیر خصوصی Balance جمع می‌کنیم.
2. **WithdrawAmount** : در این متد، سه قطعه منطق نوشته‌ایم. اول، بررسی می‌کنیم که آیا مبلغ برداشت کمتر از موجودی موجود است یا خیر. دوم، بررسی می‌کنیم که آیا مبلغ برداشت از محدودیت برداشت روزانه بیشتر است یا خیر. در صورتی که این دو شرط برقرار باشند، مبلغ را برداشت کرده و مقدار true را به کاربر برمی‌گردانیم.
3. **CheckBalance:** پیاده‌سازی این متد بسیار سرراست است؛ هر مقداری که در متغیر Balance داشته باشیم، به سادگی آن را برمی‌گردانیم.

کد کامل کلاس در زیر آمده است.

```csharp
using System;

namespace BankApplication
{
    public class SavingAccount : IBankAccount
    {
        private decimal Balance = 0;
        private readonly decimal PerDayWithdrawLimit = 10000;
        private decimal TodayWithdrawal = 0;

        public bool DepositAmount(decimal Amount)
        {
            Balance = Balance + Amount;
            Console.WriteLine($"You have Deposited: {Amount}");
            Console.WriteLine($"Your Account Balance: {Balance}");
            return true;
        }

        //Maximum Withdrawal Per Day: 10000
        public bool WithdrawAmount(decimal Amount)
        {
            if (Balance < Amount)
            {
                Console.WriteLine("You have Insufficient balance!");
                return false;
            }
            else if (TodayWithdrawal + Amount > PerDayWithdrawLimit)
            {
                Console.WriteLine("Withdrawal attempt failed!");
                return false;
            }
            else
            {
                Balance = Balance - Amount;
                TodayWithdrawal = TodayWithdrawal + Amount;
                Console.WriteLine($"You have Successfully Withdraw: {Amount}");
                Console.WriteLine($"Your Account Balance: {Balance}");
                return true;
            }
        }
        public decimal CheckBalance()
        {
            return Balance;
        }
    }
}
```

##### **حساب جاری:**

سپس، یک فایل کلاس با نام CurrentAccount.cs ایجاد کنید و سپس کد زیر را کپی و جایگذاری کنید. در اینجا، می‌توانید ببینید که ما یک متغیر خصوصی به نام Balance برای نگهداری موجودی حساب کاربر ایجاد می‌کنیم. در اینجا، ما متغیرهای PerDayWithdrawLimit و TodayWithdrawal را ایجاد نمی‌کنیم زیرا هیچ محدودیتی برای مبلغ برداشت از یک حساب جاری وجود ندارد. سپس کلاس CuurentAccount ما رابط IBankAccount را پیاده‌سازی می‌کند و پیاده‌سازی هر سه متد انتزاعی که در داخل رابط IBankAccount تعریف شده‌اند را فراهم می‌کند.

1. **DepositAmount:** این متد مبلغ واریزی را دریافت می‌کند و منطق آن مشابه SavingAccount است. هر مبلغی که کاربر ارسال می‌کند، ما فقط آن را با متغیر خصوصی Balance جمع می‌کنیم.
2. **WithdrawAmount** : در این متد، ابتدا بررسی می‌کنیم که آیا مبلغ برداشت کمتر از موجودی موجود است یا خیر. اگر مبلغ برداشت کمتر از موجودی حساب باشد، مبلغ را برداشت کرده و مقدار true را به کاربر برمی‌گردانیم.
3. **CheckBalance:** پیاده‌سازی این متد مشابه SavingAccount است؛ هر مقداری که در متغیر Balance داشته باشیم، صرفاً آن را برمی‌گردانیم.

namespace BankApplication

        public bool WithdrawAmount(decimal Amount)
        {
            if (Balance < Amount)
            {
                Console.WriteLine("You have Insufficient balance!");
                return false;

            else
            {
                Balance = Balance - Amount;
                Console.WriteLine($"You have Successfully Withdraw: {Amount}");
                Console.WriteLine($"Your Account Balance: {Balance}");

حالا، متد Main از کلاس Program را به صورت زیر تغییر دهید تا عملکرد آن را آزمایش کنید.

```csharp
using System;

namespace BankApplication
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Saving Account:");
            IBankAccount savingAccount = new SavingAccount();
            savingAccount.DepositAmount(2000);
            savingAccount.DepositAmount(1000);
            savingAccount.WithdrawAmount(1500);
            savingAccount.WithdrawAmount(5000);
            Console.WriteLine($"Saving Account Balanace: {savingAccount.CheckBalance()}");

            Console.WriteLine("\nCurrent Account:");
            IBankAccount currentAccount = new CurrentAccount();
            currentAccount.DepositAmount(500);
            currentAccount.DepositAmount(1500);
            currentAccount.WithdrawAmount(2600);
            currentAccount.WithdrawAmount(1000);
            Console.WriteLine($"Current Account Balanace: {currentAccount.CheckBalance()}");
            Console.ReadLine();
        }
    }
}
```
